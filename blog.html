1. What is DOM and What is the purpose of DOM?

DOM means Document Object Model.
The purpose of DOM is to create, change, or remove elements from the document and also to make html page more dynamic by adding events to these elements.

2. How will you select HTML elements using DOM? Name the DOM methods?

For selecting HTML elements using DOM, we can use a veriety of methods depending on specific elements.
DOM methods are, 1. document.getElementById
2. document.getElementsByClassName
3. document.getElementsByTagName
4. document.querySelector
5. document.querySelectorAll

3.What is event bubble?

The Event bubbling is a type of event propagation. The event triggers the innermost target element and consecutively triggers the parent element of the target element in the same hierarchy until it triggers the outermost element.

4.What is a callback function and why will you use it?

A callback function in javascript is a function that is passed as an argument in another function.
We use callback function to call/ execute later after its parent function(the function in which it is passed as an argument) is done executing.
